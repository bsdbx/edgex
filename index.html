EdgeX: The Ultimate Blender 3D Add-On

EdgeX is a powerful and intuitive interface Add-on for Blender 3D that enhances your workflow with specialized pie menus. Designed to extend the vanilla functionality, EdgeX boosts your performance and speed, providing you with easy access to essential tools. With utilities like Quick Crease and Bevel Weight, edge marking features, and more, EdgeX streamlines your work and saves time.

Features

Attribute Set

Quickly add or remove Crease or Bevel Weight. Adjust attributes and values precisely.

Quick Attribute

Change Crease or Bevel Weight via a property. Supports math expressions.

Attribute Select

Select by Crease or Bevel Weight using criteria: Equal, Not Equal, Greater Than, etc.

Quick Mark

Assign/remove edge marks: Sharp, Seam, Freestyle.

Mark Select

Select edges by marking type or angle: Sharp, Seam, Freestyle, or none.
